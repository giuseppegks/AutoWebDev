Aurora — ambient audio
======================

The homepage has a "Sfeer" toggle button in the bottom-right corner.
When clicked, it plays this file:

    audio/ambient.mp3

Drop your own legally-licensed spa/ambient track in this folder with
exactly that filename and it will start working — no code changes
needed.

Specs:
  - Format:  MP3, 128 kbps stereo (smaller file = faster first play)
  - Length:  3-10 minutes; the player loops automatically
  - Style:   ambient, slow, no vocals, no sudden peaks
  - Volume:  the player ducks output to ~28%, so master can be normal

Where to get a free, commercially-licensed track:

  Pixabay Music (no attribution required, free for commercial use)
    https://pixabay.com/music/search/spa/
    https://pixabay.com/music/search/relaxing/

  Mixkit (free for commercial use, no attribution)
    https://mixkit.co/free-stock-music/tag/relaxing/
    https://mixkit.co/free-stock-music/tag/spa/

  YouTube Audio Library (free, commercial OK)
    https://www.youtube.com/audiolibrary

  Free Music Archive (filter by CC0 or CC-BY)
    https://freemusicarchive.org/

DO NOT use Spotify rips, copyrighted soundtracks, or anything you do
not have a license for. The website credits MG Visuals — don't put
the project at risk.

After you save the file, hard-refresh the homepage once
(Ctrl+Shift+R) to pick it up.
